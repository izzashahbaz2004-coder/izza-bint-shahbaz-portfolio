The QA workflow

01
Requirement Analysis
Understanding functional & non-functional requirements to define the scope of testing.
02
Test Planning
Defining strategy, scope, resources, schedule and risk management.
03
Test Case Design
Writing detailed, structured test cases covering all positive and negative scenarios.
04
Execution
Running test cases systematically across all features and environments.
05
Bug Reporting
Logging clear, reproducible defects with priority classification in JIRA.
06
Retesting & Sign-off
Verifying bug fixes, performing regression, and confirming quality sign-off.
